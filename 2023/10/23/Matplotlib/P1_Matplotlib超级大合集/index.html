P1_Matplotlib超级大合集

发表于 2023-10-23 更新于 2023-10-25 分类于 Matplotlib

绘制x点和y点

  • 使用 plt.plot 函数，传入两个点的坐标默认绘制一条线

无线绘图：

  • 仅绘制标记，使用快捷字符号参数’o’

多点：

  • 根据需要绘制任意数量的点，只要确保两个轴上的点数相同即可

默认x点:

  • 如果不指定x轴上的点，x轴坐标将会使用默认值0,1,2,3…..

格式化字符串fmt：

  • 使用快捷字符号参数来指定标记
  • 此参数也称为fmt，使用以下语法来编写：
    marker,line,color

线条

线型：

  • 使用关键字参数 linestyle 或者 ls 来更改绘制线的样式：

线型更短的语法：

  • 线条原始可以用更短的语法编写：
    • linestyle 可以写成 ls
    • dotted 可以写成 :
    • dashed 可以写成 –

线条颜色：

  • 使用关键字参数 color 或 c 来设置线条的颜色：
    • 可以使用快捷字符码也可以使用十六进制或颜色名

线宽度：

  • 使用关键字参数 linewidth 或 lw 来更改线条的宽度
    • 参数是一个浮点数，单位为磅

多行：

  • 通过简单地添加更多 plt.plot() 函数来绘制任意数量的线：
  • 还可以同时添加每条线的xy坐标，只需要使用一个函数，将xy坐标成对写入就可以 (图像同上)

标签

为绘图创建标签：

  • 使用 xlabel() 和 ylabel() 函数为x轴与y轴设置标签

为绘图创建标题：

  • 使用 title() 函数为绘图设置标题

设置标题和标签的字体属性：

  • 使用 xlabel()、ylabel() 和 title() 中的 fontdict 参数来设置标题和标签的字体属性。
    （字体原因没有显示出来）

定位标题：

  • 使用 title() 函数中的 loc 参数来定位标题的位置
  • 合法值是：”left”、”right” 和 “center”

网格线

向绘图中添加网格线：

  • 使用 grid() 函数将网格线添加到绘图中

指定要显示的网格线：

  • 使用 grid() 函数中的轴参数来指定要显示的网格线
  • 合法值为：”x”、”y” 和 “both” ,默认值为 “both”

设置网格的线属性：

  • 自定义网格线的线条属性

多图

subplot显示多个图：

  • 使用 subplots() 函数，可以在一张图中绘制多个图:
  • 参数：subplot(<几行子图>， <几列子图>， <第几张子图>)
  • 两行一列示例：

子图标题：

  • 子图也可以使用 title() 函数添加标题

超级标题：

  • 使用 suptitle() 函数为多张子图添加超级标题

散点图

创建散点图：

  • 使用 scatter() 函数会值散点图
  • 船传入参数需要两个长度相同的数组，一个用作x轴的值，一个用作y轴的值

颜色：

  • 使用 color 或 c 参数为每个散点图设置自己的颜色

为每个点上色：

  • color 参数传入一个数组，可以对分别为每个点上色

颜色图：

  • 使用 cmap 参数指定颜色图，在使用颜色图的散点图中，此外还需要创建一个 0-100 取值范围内的数组。

尺寸：

  • 使用 s 参数更改点的大小
  • 就像颜色一样，需要确保大小数组的长度与x和y轴的数组长度相同

透明度：

  • 使用 alpha 参数可以调整点的透明度
  • 就像颜色一样，需要确保大小数组的长度与x和y轴的数组长度相同

组合颜色大小透明度：

柱状图

创建柱状图：

  • 使用 bar() 函数绘制柱状图
  • 另一种参数设置方法

水平柱状图：

  • 要创建水平形态的柱状图，需使用 barh() 函数

柱状图颜色：

  • bar( ) 或 barh () 函数，都可以使用 color 参数来设置颜色：

线条宽度：

  • bar() 函数生成的竖型柱状图要想设置柱状图宽度，可以使用 width 参数

线条高度：

  • barh() 函数成成的横型柱状图要想设置柱状图高度，可以使用 hight 参数

直方图

创建直方图：

  • 使用 hist() 函数来创建直方图，参数为一个数字数组

饼图

创建饼图：

  • 使用 pie() 函数创建饼图：

标签：

  • 使用 label 参数为饼图添加标签
  • label必须是一个数组，每个楔形都有一个标签

起始角度：

  • 饼图默认起始角位于x轴，若要自定义起始角度可以使用 startangle 参数来设置
  • startangle 参数的单位为度，默认值为0度,从x轴正半轴向逆时针计算角度

Explode：

  • 使用 explode 参数使饼图其中一个楔子脱离饼图，着重显示
  • explode 参数的值为一个数组，每个数组元素对应一个楔子，总数需要相等
  • 数组中的每个元素为浮点型，表示对应楔子距离圆心有多远

阴影：

  • 通过 shadow 参数为 True 为饼图添加阴影

颜色：

  • 使用 colors 参数设置每个楔形的颜色
  • 如果指定了颜色参数，则必须传入一个数组，对应每个楔形的值

图例：

  • 使用 legend() 函数为每个楔形添加解释列表
