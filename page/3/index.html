P27_顶帽运算与黑帽运算

发表于 2023-10-30 分类于 OpenCV

顶帽操作

  • 顶帽 = 原图 - 开运算
  • 开运算的效果是去除图像外的噪点，原图 - 开运算就得到了去掉的噪点.

黑帽操作

  • 黑帽 = 原图 - 闭运算
  • 闭运算可以将图像内部的噪点去掉，那么原图 - 闭运算的结果就是图型内部的噪点.

P26_形态学梯度

发表于 2023-10-30 分类于 OpenCV

形态学梯度

  • 梯度 = 原图 - 腐蚀
  • 腐蚀之后原图的边缘变小了，原图 - 腐蚀就看见恶意得到腐蚀掉的部分，即边缘.

P25_膨胀操作与开闭运算

发表于 2023-10-30 分类于 OpenCV

膨胀操作

  • 膨胀是腐蚀的相反操作，基本原理是只要保证卷积核的锚点是非0值，周边无论是0还是非0值，都变成非0值.
  • dilate(img, kernel, iterations=1)

开运算

  • 开运算和闭运算都是腐蚀和膨胀的基本应用.
  • 开运算 = 腐蚀 + 膨胀
  • morphologyEx(img, MORPH_OPEN,kernel)
    • MORPH_OPEN：表示形态学开运算
    • kernel：如果噪点比较多，会选择大一点的kernel，如果噪点比较小，可以选择小点的kernel

闭运算

  • 闭运算 = 膨胀 + 腐蚀

P24_腐蚀操作与获取形态学卷积核

发表于 2023-10-30 分类于 OpenCV

腐蚀操作

  • 腐蚀操作也就是用卷积核扫描图像，只不过腐蚀操作的卷积核一般都是1，如果卷积核内所有像素都为白色，那么锚点即为白色.
  • 大部分时候腐蚀操作使用的都是为全为1的卷积核.
  • erode(src, kernel[, dst[, anchor[, iterations[, borderType[, borderValue]]]]])
    • iterations是腐蚀操作的迭代次数，次数越多，腐蚀操作执行的次数越多，腐蚀效果越明显

获取形态学卷积核

  • opencv提供了获取卷积核的api函数，不需要手工创建卷积核.
  • getStructuringElement(shape, ksize[, anchor])
    • shape是卷积核的形状，注意不是长宽，是卷积核中1形成的形状.
    • MORTH_RECT：卷积核中的1是矩形，最常用
    • MORTH_ELLIPSE：椭圆
    • MORTH_CROSS：十字

P23_自适应阈值二值化

发表于 2023-10-30 分类于 OpenCV

自适应阈值二值化

  • 在前面的部分我们使用的是全局阈值，整幅图像采用同一个数作为阈值；这种方法并不适用于所有情况，尤其是当同一幅图像上的不同部分具有不同亮度时。这种情况下我们需要采用自适应阈值。此时的阈值是根据图像上的每一小区域计算预期对应的阈值。因此在同一幅图像上的不同区域采用的是不同的阈值，从而使我们能在亮度不同的情况下得到更好的结果。
  • adaptiveThreshold(src, maxValue, adaptiveMethod, thresholdType, blockSize, C, dst=None)
    • 这种方法需要我们指定6个参数，返回值只有一个。
    • Adaptive Method：指定计算阈值的方法
      • cv2.ADPTIVE_THRESH_MEAN_C：阈值取自相邻区域的平均值.
      • cv2.ADPTIVE_THRESH_GAUSSIAN_C：取值取值相邻区域的加权和，权重为一个高斯窗口.
    • BlockSize：邻域大小（用来计算阈值的区域大小）
    • C：这既是一个常数，阈值就等于的平均值或者加权平均值减去这个常数。、

形态学概述

  • 什么是形态学
    • 指一些列处理图像形状特征的图像处理技术。
    • 形态学的基本思想是利用一种特殊的结构元（本质上就是卷积核）开测量或提取输入图像中相应的形状或特征，以便进一步进行图像分析和目标识别.
    • 这些处理方法基本是对二进制图像进行处理，即黑白图像
    • 卷积核决定着图像处理后的效果
    • 形态学常用的基本操作有：
      • 膨胀和腐蚀
      • 开运算
      • 闭运算
      • 顶帽
      • 黑帽

图像全局二值化

P21_Canny边缘检测

发表于 2023-10-30 分类于 OpenCV

边缘检测Canny

  • Canny边缘检测算法是 John F. Canny 于 1988 年开发出来的一个多级边缘检测算法，也被很多人认为是边缘检测的最优算法，最优边缘检测的三个主要评价是：
    • 低错误率：标识出尽可能多的实际边缘，同时尽可能的减少噪音产生的误报。
    • 搞定位性：标识出的边缘要与图像中的实际边缘尽可能接近。
    • 最小响应：图像中的边缘中能标识一次。
  • Canny边缘检测的一般步骤：
    • 去噪：边缘检测容易收到噪声影响，在进行边缘检测前通需要先进行去噪，一般用高斯滤波去除噪声.
    • 计算梯度：对平滑后的图像采用Sobel算子计算梯度和方向.
      • $G = \sqrt{G_x^2 + G_y^2}$为了方便一般可以改用绝对值.
      • $\theta = arctan(\frac{G_y}{G_x})$
      • 梯度的方向被归类为四类：垂直，水平和两个对角线.

      • 非极大值抑制
        • 在获取了梯度和方向后，便利图像，去除所有不是边界的点

P20_拉普拉斯算子

发表于 2023-10-30 分类于 OpenCV

拉普拉斯算子

  • 索贝尔算子是模拟一阶求导，导数越大的地方说明变化越剧烈，越有可能是边缘.

  • 那如果继续对$f’(t)$求导呢？

  • 我们可以发现“边缘处的二阶导数=0”，可以利用这一特性去寻找图像的边缘.注意有一个问题，二阶求导的0的位置也可能是无意义的位置.

  • 拉普拉斯算子的推导过程：

    • 以x方向求解为例：
      一阶差分：$f’(x) = f(x) - f(x-1)$
      二阶差分：$f’’(x) = f’(x+1) - f’(x) = (f(x+1)-f(x))-(f(x)-f(x-1))$
      化简后：$f’’(y)=(f(y-1)-2f(y))+f(y+1)$

      把x,y方向的梯度叠加在一起.
      $f’’(x, y) = f’_x(x, y)+f’_y(x,y)$
      $f’’(x, y) = f(x-1, y)-2f(x, y)+f(x+1,y)+f(x,y-1)-2f(x,y)+f(x, y+1)$
      $f’’(x, y) = f(x-1, y)+f(x+1,y)+f(x,y-1)+f(x,y+1)-4f(x,y)$

      这个等式可以写成:
      $$
      f’’(x,y) = \begin{bmatrix}
      0 & 1 & 0 \newline
      1 & -4 & 1 \newline
      0 & 1 & 0 \end{bmatrix}
      \odot
      \begin{bmatrix}
      f(x-1, y-1) & f(x, y-1) & f(x+1, y-1) \newline
      f(x-1, y) & f(x, y) & f(x + 1, y) \newline
      f(x-1, y+1) & f(x,y+1) & f(x+1, y+1) \end{bmatrix}
      $$
      这样就得到了拉普拉斯算子的卷积核即卷积模板.

  • Laplacian(src, ddepth[, dst[,dst[, ksize[, scale[, delta[, borderType]]]]]])

  • 可以同时求两个方向的边缘.

  • 对噪音敏感，一般需要先进行去噪再调用拉普拉斯

P19_沙尔(Scharr)算子

发表于 2023-10-30 分类于 OpenCV

沙尔(Scharr)算子

  • Scharr(src, ddepth, dx, dy[, dst[, scale[, delta[, borderType]]]])
  • 当内核大小为3时，以上Sobel内核可能产生比较明显的误差（毕竟Sobel算子只是求取了导数的近似值）。为了解决这一问题，OpenCV提供了Scharr函数，但该函数仅作用于大小为3的内核。该函数的运算与Sobel函数一样快，但结果更加精确.
  • Scharr算子和Sobel很类似，只不过使用了不同的Kernel值，放大了像素变化的情况：
  • Scharr算子的特点：
    • 只支持3*3大小的kernel，所以没有kernel参数.
    • 只能求x方向或y方向的边缘.
    • 把Sobel算子的ksize设为-1就是Scharr算子.
    • Scharr算子擅长寻找细小的边缘，一般用的较少.

P18_索贝尔(sobel)算子

发表于 2023-10-29 更新于 2023-10-30 分类于 OpenCV

索贝尔(sobel)算子

  • 前面讲的各种滤波主要功能都是去除噪声，这样的滤波统称为高通滤波；而接下里要将的各种算子，主要是用来寻找边缘，统称为低通滤波.
  • 边缘是像素值发生跃迁的位置，是图像的显著特征之一，在图像特征提取，对象检测，模式识别等方面都有重要的作用.
  • 人眼如何识别边缘？
    • 比如有一幅图，图里面有一条线，左边很亮，右边很暗，那人眼就容易识别这条线作为边缘.也就是像素的灰度值快速变化的地方.
  • sobel算子对于图像求一阶导数。一阶导数越大，说明像素在该方向的变化越大，边缘信号越强。
  • 因为图像的灰度值都是离散的数字，sobel算子采用离散差分算子计算图像像素点值的近似梯度.
  • 图像是二维的，即沿着宽度/高度两个方向.
  • 我们使用两个卷积核对原图像进行处理：
    • 水平方向
    • 垂直方向
  • 这样，我们就得到了两个新的矩阵，分别反映了每一点像素在水平方向的亮度变化情况和在垂直方向上的亮度变化情况.
  • 综合考虑这两个反向的变化，我们使用以下公式反应某个像素的梯度变化情况.
    $$
    G = \sqrt{G_x^2 + G_y^2}
    $$
  • 有时候为了简单起见，也直接用绝对值相加替代：$G = |G_x|+|G_y|$
