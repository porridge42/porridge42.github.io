P72集合框架_集合概念

发表于 2022-11-22 分类于 JavaSE

什么是集合

  • 概念：对象的容器，定义了对多个对象进行操作的常用方法。可实现数组的功能。


  • 和数组的区别：

    1. 数组长度固定，集合长度不固定。
    2. 数组可以存储基本类型和引用类型，集合只能存储引用类型。

  • 位置:

    • java.util.*;

P71常用类_常用类小结

发表于 2022-11-21 更新于 2022-11-22 分类于 JavaSE

常用类总结

  • 内部类：
    • 在一个类的内部再定义一个完整的类。
    • 成员内部类、静态内部列、局部内部类、匿名内部类。
  • Object类：
    • 所有类的直接或间接父类，可存储任何对象。
  • 包装类：
    • 基本数据类型所对应的引用类型，可以使用Object统一所有数据。
  • String类：
    • 字符串是常量，创建以后不可改变，字面值保存在字符串池中，可以共享。
  • BigDecimal类：
    • 可以精确计算浮点数。
  • Date类：
    • 特定时间。
    • 已经过时，部分方法已弃用。
  • Calendar类：
    • Date类的改进版。
  • SimpleDateFormat类：
    • 格式化转化时间对象。
  • System类：
    • 系统类。

P63常用类_String类常用方法

发表于 2022-11-13 分类于 JavaSE

常用类方法

  • int length()
    • 返回字符串的长度
  • char charAt(int index)
    • 返回某个位置的字符
  • boolean contains(String str)
    • 判断当前字符串是否包含指定字符串
  • char[] toCharArray()
    • 将字符串转换为字符数组
  • int indexOf(String str)
    • 查找指定字符串首次出现的位置，若不存在则返回-1
  • int lastIndexOf(String str)
    • 查找指定字符串最后一次出现的位置，若不存在则返回-1
  • String trim()
    • 去掉字符串前后的空格
  • String toUpperCase()
    • 将小写转换成大写
  • boolean endWith(String str)
    • 判断字符串是否以Str结尾
  • String repace(charn,bchar)
    String replace(CharSequence, CharSequence)
    • 替换字符串中的一段字符或字符串
  • String[] split(String str)
    • 根据str堆字符串进行拆分
