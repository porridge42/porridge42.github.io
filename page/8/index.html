List的使用2

  • 添加数字数据
    • List集合在添加数字数据的时候会进行自动装箱操作，将基本类型的数字数据自动装箱为对相应的包装类对象。
  • 删除数字数据
    • List集合在删除数字数据的时候需要使用引用类型来指定操作元素，如使用基本类型数字则表示删除指定下标的数据。
  • SubList()方法
    • List SubList(int fromIndex，int toIndex)
    • 用于返回一个子集合。
    • 子集合截取的范围从参数fromIndex指定的下标到参数toIndex所指定的下标，遵循含头不含尾规则，即包含下标为fromIndex的元素但不包含下标为toIndex的元素。

P72集合框架_集合概念

发表于 2022-11-22 分类于 JavaSE

什么是集合

  • 概念：对象的容器，定义了对多个对象进行操作的常用方法。可实现数组的功能。


  • 和数组的区别：

    1. 数组长度固定，集合长度不固定。
    2. 数组可以存储基本类型和引用类型，集合只能存储引用类型。

  • 位置:

    • java.util.*;

P71常用类_常用类小结

发表于 2022-11-21 更新于 2022-11-22 分类于 JavaSE

常用类总结

  • 内部类：
    • 在一个类的内部再定义一个完整的类。
    • 成员内部类、静态内部列、局部内部类、匿名内部类。
  • Object类：
    • 所有类的直接或间接父类，可存储任何对象。
  • 包装类：
    • 基本数据类型所对应的引用类型，可以使用Object统一所有数据。
  • String类：
    • 字符串是常量，创建以后不可改变，字面值保存在字符串池中，可以共享。
  • BigDecimal类：
    • 可以精确计算浮点数。
  • Date类：
    • 特定时间。
    • 已经过时，部分方法已弃用。
  • Calendar类：
    • Date类的改进版。
  • SimpleDateFormat类：
    • 格式化转化时间对象。
  • System类：
    • 系统类。
